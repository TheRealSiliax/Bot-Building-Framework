# 07_test - Testdokumentation

Dieser Ordner enthält die Testdokumentation für den Lernbot.

## Inhalt

| Datei | Beschreibung |
|-------|--------------|
| `testplan.md` | Testplan mit Szenarien |
| `testergebnisse.md` | Dokumentierte Testergebnisse |

## Testablauf

1. **Testplan erstellen** - Definiere Testszenarien
2. **Tests durchführen** - Führe Szenarien durch
3. **Ergebnisse dokumentieren** - Halte Ergebnisse fest
4. **Bei Bedarf: Iteration** - Zurück zum entsprechenden Schritt

## Test-Kategorien

- **Happy Path**: Normaler, erwarteter Ablauf
- **Edge Cases**: Grenzfälle und ungewöhnliche Eingaben
- **Error Handling**: Reaktion auf Fehler
- **Robustness**: Widerstandsfähigkeit gegen Manipulation

## Vorlage

Nutze `testplan_template.md` aus `templates/scripts/` als Vorlage.
