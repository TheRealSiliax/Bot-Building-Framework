# Reflexionszyklus

Ein strukturierter Prozess für regelmäßige Selbstreflexion, der Selbsterkenntnis fördert und kontinuierliches Lernen ermöglicht.

## Überblick

```
     ┌────────────────────────────────────────────┐
     │                                            │
     ▼                                            │
 ERLEBEN → BEOBACHTEN → ANALYSIEREN → INTEGRIEREN ┘
     │
     │ (fortlaufend)
     ▼
 HANDELN (informiert durch Reflexion)
```

## Beteiligte Agenten
- **Hauptagent**: Reflektor
- **Unterstützend**: Coach (bei Blockaden), Researcher (bei Fragen)

## Wann welche Reflexion nutzen?

```
┌─────────────────────────────────────────────────────────────┐
│ Was brauchst du?                        → Welche Reflexion? │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ "Kurzer täglicher Check-In"            → Tägliche Mikro    │
│ "Etwas Bedeutsames ist passiert"       → Ereignis-Reflexion│
│ "Die Woche Revue passieren lassen"     → Wöchentlicher Rev.│
│ "Quartals-/Jahresrückblick"            → Tiefenreflexion   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Reflexions-Varianten

### 1. Tägliche Mikro-Reflexion (5-10 Min)
> Kurzer Check-In am Ende des Tages.

**→ Vollständige Anleitung:** `sops/daily_reflection.md`
**→ Template:** `templates/reflection_journal.md` (Abschnitt "Tägliche Mikro-Reflexion")

**Kernelemente:**
- 3 gute Dinge (Gratitude)
- Highlight & Challenge
- Ein Learning
- Morgen-Intention

---

### 2. Ereignis-Reflexion (15-30 Min)
> Nach bedeutsamen Situationen oder Erlebnissen.

**→ Methode:** Gibbs Reflective Cycle (siehe `agents/reflektor.md`)
**→ Template:** `templates/reflection_journal.md` (Abschnitt "Ereignis-Reflexion")

**Wann anwenden:**
- Nach wichtigen Gesprächen
- Nach Konflikten
- Nach Erfolgen oder Misserfolgen
- Bei starken emotionalen Reaktionen

**Die 6 Phasen:**
1. Beschreibung → Was ist passiert? (Fakten)
2. Gefühle → Was habe ich gefühlt?
3. Bewertung → Was war gut/schlecht?
4. Analyse → Warum ist es so gelaufen?
5. Schlussfolgerung → Was hätte ich anders machen können?
6. Aktionsplan → Was werde ich nächstes Mal tun?

---

### 3. Wöchentlicher Review (30-45 Min)
> Umfassender Rückblick und Vorausplanung.

**→ Vollständige Anleitung:** `sops/weekly_review.md`
**→ Template:** `templates/reflection_journal.md` (Abschnitt "Wöchentlicher Review")

**Kernelemente:**
- Daten/Tracking sammeln
- Wins würdigen
- Muster erkennen
- Werte-Alignment prüfen
- Nächste Woche planen

---

### 4. Quartals-/Tiefenreflexion (60-90 Min)
> Umfassende Analyse über längeren Zeitraum.

**→ Template:** `templates/reflection_journal.md` (Abschnitt "Quartals-Review")

**Kernelemente:**
- Zielerreichung bewerten
- Trends über Zeit analysieren
- Highlight-Reel erstellen
- Werte-Entwicklung reflektieren
- Nächstes Quartal planen

## Tipps für nachhaltige Praxis

### Ritualisieren
- Feste Zeit, fester Ort
- Nach bestehender Gewohnheit einbauen (Habit Stacking)
- Timer stellen

### Nicht übertreiben
- Lieber 5 Min täglich als 1h unregelmäßig
- Bei Zeitmangel: Nur "3 gute Dinge"
- Qualität > Quantität

### Ehrlich sein
- Niemand liest das außer dir
- Unangenehmes nicht ausklammern
- Selbstmitgefühl üben

### Muster nutzen
- Regelmäßig zurückblättern
- Muster markieren
- Erkenntnisse in Knowledge Base speichern
