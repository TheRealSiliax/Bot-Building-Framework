# Prozesse — Übersicht

Die Prozesse definieren, wie die verschiedenen Agenten zusammenarbeiten und in welcher Reihenfolge Entwicklungsschritte ablaufen.

## Verfügbare Prozesse

### 1. Reflexionszyklus (`reflection_cycle.md`)
Tägliche/wöchentliche Selbstreflexion mit dem Reflektor-Agent.
- **Wann nutzen**: Regelmäßige Selbstreflexion, nach wichtigen Ereignissen
- **Dauer**: 10-30 Minuten
- **Frequenz**: Täglich (kurz) oder wöchentlich (ausführlich)

### 2. Spiral-Breaker (`spiral_breaker.md`)
Systematisches Erkennen und Durchbrechen von Negativspiralen.
- **Wann nutzen**: Bei wiederkehrenden destruktiven Mustern
- **Beteiligte Agenten**: Pattern-Analyst → Researcher → Stratege
- **Dauer**: 45-60 Minuten (einmalige Analyse)

### 3. Evidenz-Intervention (`evidence_intervention.md`)
Wissenschaftlich fundierte Intervention für spezifische Herausforderungen.
- **Wann nutzen**: Bei konkretem Problem, das gelöst werden soll
- **Beteiligte Agenten**: Researcher → Stratege → Coach
- **Dauer**: Variable

### 4. Growth-Loop (`growth_loop.md`)
Kontinuierlicher Verbesserungszyklus mit regelmäßigen Reviews.
- **Wann nutzen**: Für langfristige Ziele und Gewohnheitsaufbau
- **Beteiligte Agenten**: Alle, je nach Phase
- **Dauer**: Ongoing

## Prozess-Auswahl

```
┌─────────────────────────────────────────────────────────┐
│ Was brauchst du gerade?                                 │
├─────────────────────────────────────────────────────────┤
│                                                         │
│ "Ich will verstehen, was in mir vorgeht"               │
│ → Reflexionszyklus                                      │
│                                                         │
│ "Ich stecke in einem wiederkehrenden Muster fest"      │
│ → Spiral-Breaker                                        │
│                                                         │
│ "Ich habe ein konkretes Problem zu lösen"              │
│ → Evidenz-Intervention                                  │
│                                                         │
│ "Ich will langfristig wachsen und Ziele erreichen"     │
│ → Growth-Loop                                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
