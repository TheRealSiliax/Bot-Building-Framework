## Phase 1: ASSESS (Standortbestimmung)

### Ziel
Klare Standortbestimmung über alle drei Führungsebenen hinweg.

### Beteiligte Agenten
- **Reflektor** → Selbsteinschätzung
- **Leader** → Strukturierte Bewertung
- **Researcher** → Evidenzbasierte Assessment-Tools

### Methoden

#### 1. Selbst-Assessment
Bewertung aller 15 Kernkompetenzen (je 5 pro Ebene):

```
Leading Self         │ Leading Others      │ Leading Organization
─────────────────────┼─────────────────────┼─────────────────────
Selbstbewusstsein    │ Kommunikation       │ Vision & Strategie
Selbstregulation     │ Motivation          │ Strategisches Denken
Persönliches Wachstum│ Entwicklung anderer │ Change Management
Accountability       │ Team-Building       │ Kultur-Building
Werte & Purpose      │ Konfliktlösung      │ Ressourcenmanagement
```

#### 2. 360°-Feedback (empfohlen)
Feedback von:
- Vorgesetzten
- Peers
- Direkten Mitarbeitern
- Anderen Stakeholdern

#### 3. Gap-Analyse
- Größte Stärken identifizieren
- Kritischste Entwicklungsfelder finden
- Priorisierung nach Hebelwirkung

### Output
- Vollständiges Leadership-Assessment
- Priorisierte Liste von Entwicklungsfeldern
- Identifizierte Stärken als Ressourcen

---

## Phase 2: DEFINE (Purpose & Vision klären)

### Ziel
Klarheit über das persönliche "Warum" als Führungskraft und die angestrebte Führungsidentität.

### Beteiligte Agenten
- **Coach** → Kraftvolle Fragen
- **Reflektor** → Werte-Exploration

### Kernfragen

**Purpose:**
- Warum will ich führen?
- Welche Wirkung will ich haben?
- Was treibt mich an?

**Vision:**
- Welche Führungskraft will ich sein?
- Wie sollen Menschen über meine Führung sprechen?
- Was soll mein Leadership-Erbe sein?

**Identität:**
- "Ich bin eine Führungskraft, die..."
- Welche Werte leiten meine Führung?
- Wofür stehe ich?

### Output
- Führungs-Purpose Statement
- Führungsvision
- Führungs-Identitäts-Statement

---

## Phase 3: STRATEGIZE (Entwicklungsplan erstellen)

### Ziel
Konkreten, umsetzbaren Entwicklungsplan für 4-8 Wochen erstellen.

### Beteiligte Agenten
- **Stratege** → Taktiken und Implementation Intentions
- **Leader** → Führungsspezifische Interventionen
- **Accountability** → Tracking-System

### Prozess

#### 1. Fokusbereich wählen
- NUR EINEN Bereich für den nächsten Zyklus
- Basierend auf größter Hebelwirkung
- Realistisch für den Zeitraum

#### 2. Verhaltensexperimente definieren
3 konkrete, beobachtbare Verhaltensänderungen:

```
Verhalten: [Was genau werde ich anders tun?]
Situation: [Wann/Wo werde ich es üben?]
Implementation Intention: "Wenn X, dann Y"
Frequenz: [Wie oft pro Woche?]
```

#### 3. Feedback-System einrichten
- Selbstreflexion (täglich/wöchentlich)
- Externe Feedback-Quellen
- Messbare Indikatoren

### Output
- Gewählter Fokusbereich
- 3 Verhaltensexperimente mit Implementation Intentions
- Tracking- und Feedback-System

---

## Phase 4: PRACTICE & REST (Üben und Konsolidieren)

### Ziel
Neue Führungsverhaltensweisen durch deliberate practice verankern, mit bewussten Ruhephasen zur Integration.

### Beteiligte Agenten
- **Coach** → Motivation und Reflexion
- **Accountability** → Tracking und Check-Ins
- **Reflektor** → Tiefenreflexion in Rest-Phase

### Der Practice-Rest-Zyklus

```
┌─────────────────────────────────────────────────────────────┐
│                    8-WOCHEN-ZYKLUS                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  WOCHE 1-3: INTENSIVE PRACTICE                             │
│  ─────────────────────────────────                         │
│  • Verhaltensexperimente täglich anwenden                  │
│  • Nach jeder Situation kurz reflektieren                  │
│  • Feedback aktiv einholen                                 │
│  • Anpassungen vornehmen                                   │
│                                                             │
│  WOCHE 4: REST & CONSOLIDATE                               │
│  ───────────────────────────                               │
│  • Weniger bewusste Anstrengung                           │
│  • Learnings sacken lassen                                │
│  • Tiefenreflexion durchführen                            │
│  • Erholung priorisieren                                  │
│                                                             │
│  WOCHE 5-7: CONTINUED PRACTICE                             │
│  ─────────────────────────────                             │
│  • Verfeinern und vertiefen                               │
│  • Variation einbauen                                      │
│  • Schwierigkeitsgrad erhöhen                             │
│                                                             │
│  WOCHE 8: REVIEW & TRANSITION                              │
│  ───────────────────────────                               │
│  • Fortschritt evaluieren                                 │
│  • Entscheiden: Vertiefen oder neuer Fokus               │
│  • Nächsten Zyklus planen                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Transformational Leadership Integration

### Während des gesamten Prozesses die 4 I's im Blick behalten:

| I | Frage für Reflexion |
|---|---------------------|
| **Idealized Influence** | "Handle ich als Vorbild?" |
| **Inspirational Motivation** | "Inspiriere ich durch Vision und Sinn?" |
| **Intellectual Stimulation** | "Rege ich zum Denken und Hinterfragen an?" |
| **Individualized Consideration** | "Sehe ich jeden als Individuum?" |

---

## Zyklus-Übergang

### Nach jedem 8-Wochen-Zyklus

**Evaluation:**
1. Wurde das Ziel erreicht?
2. Was hat funktioniert?
3. Was waren die größten Learnings?
4. Was ist der nächste sinnvolle Schritt?

**Entscheidung:**
□ **Vertiefen:** Gleicher Fokus, nächste Stufe
□ **Erweitern:** Verwandter Bereich, aufbauend
□ **Wechseln:** Neuer Fokus, andere Ebene
□ **Pausieren:** Konsolidierung ohne aktiven Fokus
