# SOP: Evidenzbasierte Zielsetzung (SMART+)

## Zweck
Effektive, motivierende und erreichbare Ziele setzen, die auf wissenschaftlichen Erkenntnissen basieren.

## Dauer
20-30 Minuten pro Ziel

## Wann anwenden
- Neue Ziele definieren (Quartal, Jahr)
- Bestehende Ziele überarbeiten
- Wenn Ziele nicht motivieren oder unrealistisch erscheinen
- Nach dem Scheitern eines Ziels (neu formulieren)

---

## Schritt-für-Schritt Anleitung

### Phase 1: Klarheit gewinnen (5-10 Min)

#### Schritt 1.1: Das "Warum" klären
Bevor du das "Was" definierst, verstehe das "Warum":

```
Was möchte ich erreichen?
_________________________________________

WARUM ist mir das wichtig?
_________________________________________

Welchen meiner Kernwerte bedient dieses Ziel?
□ Gesundheit    □ Beziehungen    □ Wachstum
□ Freiheit      □ Sicherheit     □ Erfolg
□ Kreativität   □ Beitrag        □ _________

Wie würde sich mein Leben verbessern?
_________________________________________
```

#### Schritt 1.2: Hindernisse antizipieren
Was könnte im Weg stehen?

```
Mögliche Hindernisse:
1. _____________________________________
2. _____________________________________
3. _____________________________________

Bin ich bereit, diese zu überwinden?
□ Ja    □ Nein → Ziel überdenken    □ Unsicher
```

---

### Phase 2: SMART+ Formulierung (10-15 Min)

#### S - Spezifisch
Das Ziel muss klar und konkret sein.

```
❌ Vage: "Ich will gesünder leben"
✅ Spezifisch: "Ich esse täglich 5 Portionen Obst/Gemüse"

Mein spezifisches Ziel:
_________________________________________

Ist es klar genug, dass jemand anderes verstehen würde?
□ Ja    □ Nein → Präzisieren
```

#### M - Messbar
Wie weißt du, ob du es erreicht hast?

```
Wie messe ich den Fortschritt?
_________________________________________

Was ist der Zielwert?
_________________________________________

Beispiel: "5 Portionen Obst/Gemüse täglich" → Tägliches Tracking

Ist das Ziel objektiv messbar?
□ Ja    □ Nein → Messkriterien definieren
```

#### A - Attraktiv (nicht nur "Achievable")
Das Ziel muss dich emotional ansprechen.

```
Wie aufgeregt bin ich bei diesem Ziel (1-10)? [ ]

Wenn unter 7: Was würde es attraktiver machen?
_________________________________________

Visualisiere: Wie fühlt es sich an, das Ziel erreicht zu haben?
_________________________________________
```

#### R - Realistisch
Ist es erreichbar mit deinen Ressourcen?

```
Habe ich die Zeit? 
Benötigt: ca. ___ Stunden/Woche
Verfügbar: ca. ___ Stunden/Woche

Habe ich die Fähigkeiten (oder kann ich sie erwerben)?
□ Ja    □ Nein, ich brauche: ________________

Habe ich die Unterstützung?
□ Ja    □ Nein, ich brauche: ________________

Ist das Ziel eine Herausforderung, aber machbar?
□ Ja    □ Zu leicht → Erhöhen    □ Zu schwer → Reduzieren
```

#### T - Terminiert
Bis wann?

```
Deadline: _______________________________

Ist das Timing realistisch?
□ Ja    □ Nein → Anpassen

Meilensteine (bei längeren Zielen):
□ _____________ bis _____________
□ _____________ bis _____________
□ _____________ bis _____________
```

#### + Identitäts-basiert (Der Bonus)
Wer willst du SEIN, nicht nur was willst du TUN?

```
❌ Verhaltensbasiert: "Ich will jeden Tag laufen"
✅ Identitätsbasiert: "Ich bin ein Läufer"

Welche Identität steckt hinter diesem Ziel?

"Ich bin jemand, der ___________________________"

Wie würde diese Person handeln?
_________________________________________
```

---

### Phase 3: Implementation Planning (5-10 Min)

#### Schritt 3.1: Implementation Intentions
Erstelle Wenn-Dann-Pläne:

```
Wenn [SITUATION/ZEIT], dann werde ich [VERHALTEN].

Beispiel: "Wenn ich morgens aufstehe und mich angezogen habe,
          dann gehe ich 15 Minuten laufen."

Mein Implementation Intention:
Wenn ____________________________________,
dann ____________________________________.
```

#### Schritt 3.2: Erste Schritte definieren
Was sind die konkreten nächsten Schritte?

```
In den nächsten 24 Stunden:
□ _______________________________________

In dieser Woche:
□ _______________________________________
□ _______________________________________

Im ersten Monat:
□ _______________________________________
□ _______________________________________
```

#### Schritt 3.3: Environment Design
Wie gestalte ich meine Umgebung, damit Erfolg leichter wird?

```
Was mache ich SICHTBAR (Cue für richtiges Verhalten)?
_________________________________________

Was mache ich UNSICHTBAR (Cue für falsches Verhalten)?
_________________________________________

Was mache ich EINFACH?
_________________________________________

Was mache ich SCHWIERIG?
_________________________________________
```

#### Schritt 3.4: Accountability
Wie stelle ich Verantwortlichkeit sicher?

```
□ Ich teile das Ziel mit: _________________
□ Ich nutze einen Tracker: ________________
□ Ich habe regelmäßige Check-ins mit: _____
□ Ich habe einen "Commitment Contract": ___
```

---

### Phase 4: Ziel-Statement schreiben

Fasse alles in einem Ziel-Statement zusammen:

```
┌─────────────────────────────────────────────────────────────┐
│ MEIN ZIEL                                                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ Ich bin jemand, der _____________________________________.  │
│                                                             │
│ Bis zum [DATUM] werde ich [MESSBARES ERGEBNIS].            │
│                                                             │
│ _________________________________________________________  │
│ _________________________________________________________  │
│                                                             │
│ Mein Warum: ____________________________________________   │
│                                                             │
│ Mein erster Schritt: ___________________________________   │
│                                                             │
│ Mein Implementation Intention:                              │
│ Wenn _________________, dann ___________________.          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Ziel-Qualitäts-Check

Prüfe dein Ziel mit dieser Checkliste:

```
□ Es ist SPEZIFISCH (klar und konkret)
□ Es ist MESSBAR (ich weiß, wann ich es erreicht habe)
□ Es ist ATTRAKTIV (ich bin motiviert, min. 7/10)
□ Es ist REALISTISCH (herausfordernd, aber machbar)
□ Es ist TERMINIERT (klare Deadline)
□ Es ist IDENTITÄTSBASIERT (wer ich sein will)
□ Ich habe einen Implementation Intention
□ Ich habe die ersten Schritte geplant
□ Ich habe Accountability eingerichtet
```

Wenn alle Häkchen gesetzt → Ziel ist bereit!

---

## Häufige Fehler vermeiden

| Fehler | Lösung |
|--------|--------|
| Zu viele Ziele gleichzeitig | Max. 3 große Ziele pro Quartal |
| Zu vage formuliert | Spezifischer werden |
| Kein echtes "Warum" | Tiefere Motivation erkunden |
| Nur Ergebnis, kein Prozess | Systemziele zusätzlich definieren |
| Keine Flexibilität | "Versionen" des Ziels (A, B, C) |
| Nur Willpower verlassen | Environment Design nutzen |

## Wissenschaftliche Basis

### Goal Setting Theory (Locke & Latham)
- Spezifische, schwierige Ziele → Bessere Leistung
- Effektstärke: d = 0.58 (mittel bis groß)
- Feedback und Commitment verstärken den Effekt

### Implementation Intentions (Gollwitzer)
- Verdoppeln bis verdreifachen Erfolgswahrscheinlichkeit
- Meta-Analyse: d = 0.65 (mittel bis groß)
- Besonders wirksam bei schwierigen Zielen

### Identity-Based Habits (Clear)
- Identität treibt Verhalten langfristig
- Jede Handlung = Votum für Identität
- Selbstbild-Konsistenz als Motivator

### Self-Determination Theory (Deci & Ryan)
- Intrinsische Motivation > extrinsische
- Autonomie, Kompetenz, Verbundenheit
- "Warum" wichtiger als "Was"
