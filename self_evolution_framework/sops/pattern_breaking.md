# SOP: Muster durchbrechen

## Zweck
Schritt-für-Schritt-Anleitung zum Erkennen und Durchbrechen eines destruktiven Verhaltensmusters.

## Dauer
45-60 Minuten (einmalige Analyse), dann ongoing

## Wann anwenden
- Du merkst, dass du immer wieder in dasselbe Muster fällst
- Ein Verhalten wiederholt sich trotz guter Vorsätze
- Du willst verstehen, warum du etwas tust (oder nicht tust)
- Bisherige Veränderungsversuche sind gescheitert

---

## Schritt-für-Schritt Anleitung

### Phase 1: Das Muster erkennen (15 Min)

#### Schritt 1.1: Konkretes Beispiel wählen
Denke an eine kürzliche Situation, in der das Muster aufgetreten ist:

```
Situation: ______________________________
Wann: __________________________________
Wo: ____________________________________
Wer war dabei? __________________________
Was genau ist passiert? _________________
________________________________________
```

#### Schritt 1.2: Weitere Beispiele sammeln
Wann ist das noch passiert? (Mind. 2 weitere)

```
Beispiel 2: _____________________________
Beispiel 3: _____________________________
Beispiel 4 (optional): __________________
```

#### Schritt 1.3: Gemeinsamkeiten finden
Was haben diese Situationen gemeinsam?

```
Ähnliche Auslöser: ______________________
Ähnliche Gedanken: ______________________
Ähnliche Gefühle: _______________________
Ähnliches Verhalten: ____________________
Ähnliche Konsequenz: ____________________
```

#### Schritt 1.4: Dem Muster einen Namen geben
> "Wie würdest du dieses Muster in 3-5 Worten benennen?"

```
Mein Muster heißt: "_____________________"
(z.B. "Die Vermeidungs-Falle", "Der Perfektionismus-Teufelskreis")
```

---

### Phase 2: Das Muster verstehen (15 Min)

#### Schritt 2.1: Die Kette aufschreiben
Zerlege das Muster in seine Bestandteile:

```
1. TRIGGER (Was löst es aus?)
   _____________________________________

2. ERSTER GEDANKE (Was denke ich?)
   "_____________________________________"

3. KÖRPER-REAKTION (Was spüre ich physisch?)
   _____________________________________

4. GEFÜHL (Was fühle ich?)
   _____________________________________

5. HANDLUNGS-IMPULS (Was will ich tun?)
   _____________________________________

6. VERHALTEN (Was tue ich dann?)
   _____________________________________

7. KURZFRISTIGE KONSEQUENZ (Was passiert sofort?)
   _____________________________________

8. LANGFRISTIGE KONSEQUENZ (Was kostet es mich?)
   _____________________________________
```

#### Schritt 2.2: Visualisiere die Spirale

```
     [TRIGGER]
         │
         ▼
    [GEDANKE]
         │
         ▼
     [GEFÜHL]
         │
         ▼
    [VERHALTEN]
         │
         ▼
   [KONSEQUENZ]
         │
         └──────────────► verstärkt den TRIGGER
```

#### Schritt 2.3: Die Funktion verstehen
Jedes Muster hatte/hat einen Zweck!

```
Was BEKOMME ich kurzfristig durch dieses Verhalten?
(z.B. Erleichterung, Kontrolle, Sicherheit, Aufmerksamkeit)
_________________________________________

Was VERMEIDE ich dadurch?
(z.B. Schmerz, Ablehnung, Versagen, Konfrontation)
_________________________________________

Welches BEDÜRFNIS steckt dahinter?
(z.B. Zugehörigkeit, Kompetenz, Autonomie, Sicherheit)
_________________________________________

Woher könnte dieses Muster STAMMEN?
(z.B. Kindheit, frühere Erfahrungen, Vorbilder)
_________________________________________
```

### Phase 3: Interventionspunkte identifizieren (10 Min)

#### Schritt 3.1: Wo kann ich eingreifen?
Markiere, wo Intervention möglich ist:

```
TRIGGER ──[1]──► GEDANKE ──[2]──► GEFÜHL ──[3]──► VERHALTEN ──[4]──► KONSEQUENZ
                                                                        │
                              [5] Neue Verstärkung ◄────────────────────┘

[1] TRIGGER modifizieren
    □ Kann ich den Trigger vermeiden?
    □ Kann ich ihn vorhersehen?
    □ Kann ich mich darauf vorbereiten?

[2] GEDANKEN umstrukturieren
    □ Kann ich den Gedanken hinterfragen?
    □ Gibt es eine alternative Sichtweise?
    □ Kann ich Distanz zum Gedanken gewinnen?

[3] GEFÜHLE regulieren
    □ Kann ich das Gefühl akzeptieren (statt bekämpfen)?
    □ Kann ich es anders regulieren?
    □ Kann ich Raum zwischen Gefühl und Reaktion schaffen?

[4] VERHALTEN ändern
    □ Kann ich ein alternatives Verhalten einsetzen?
    □ Kann ich mir Zeit verschaffen vor der Reaktion?
    □ Gibt es ein "inkompatibes" Verhalten?

[5] VERSTÄRKUNG ändern
    □ Kann ich die positive Konsequenz anders bekommen?
    □ Kann ich negative Konsequenzen spürbarer machen?
```

#### Schritt 3.2: Den besten Hebel wählen
Bewerte jeden Interventionspunkt:

| Punkt | Machbarkeit (1-5) | Wirkung (1-5) | Priorität |
|-------|-------------------|---------------|-----------|
| [1] Trigger | | | |
| [2] Gedanke | | | |
| [3] Gefühl | | | |
| [4] Verhalten | | | |
| [5] Verstärkung | | | |

**Start mit:** Hohe Machbarkeit × Hohe Wirkung

## Beispiel: Prokrastinations-Muster durchbrechen

### Phase 1: Erkennen
**Name:** "Die Aufschieberitis"
**Beispiele:** Große Projekte, Steuererklärung, schwierige E-Mails

### Phase 2: Verstehen
```
Trigger: Große, unklare Aufgabe
Gedanke: "Das ist zu viel"
Gefühl: Überforderung
Verhalten: Social Media checken
Kurzfristig: Erleichterung
Langfristig: Zeitdruck, Stress, Schuldgefühle
Funktion: Schutz vor Versagensangst
```

### Phase 3: Intervention
Gewählte Punkte:
1. **Trigger**: Aufgabe sofort zerlegen
2. **Verhalten**: 2-Minuten-Start-Regel

### Phase 4: Strategie
```
Wenn ich eine große Aufgabe bekomme,
dann zerlege ich sie sofort in 3 Teile.

Wenn ich Ablenkungsimpuls spüre,
dann arbeite ich erst noch 2 Minuten.
```

### Phase 5: Start
- Heute: Nächste Aufgabe sofort zerlegen
- Morgen: 2-Min-Regel ausprobieren
- Diese Woche: Täglich tracken

- **Habit Reversal Training** (Azrin & Nunn)
