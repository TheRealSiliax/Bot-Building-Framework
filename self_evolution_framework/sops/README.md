# SOPs — Standard Operating Procedures

Konkrete Schritt-für-Schritt-Anleitungen für häufige Situationen und Interventionen.

## Verfügbare SOPs

| SOP | Beschreibung | Dauer |
|-----|--------------|-------|
| `daily_reflection.md` | Tägliche Mikro-Reflexion | 5-10 Min |
| `weekly_review.md` | Wöchentlicher Review | 30-45 Min |
| `crisis_intervention.md` | Akute emotionale Krisen handhaben | 15-30 Min |
| `goal_setting.md` | Evidenzbasierte Zielsetzung (SMART+) | 20-30 Min |
| `pattern_breaking.md` | Destruktive Muster durchbrechen | 45-60 Min |

## Wie SOPs nutzen

1. **SOP auswählen** basierend auf aktueller Situation
2. **Schritte sequenziell durchgehen** - nicht überspringen
3. **Outputs dokumentieren** in entsprechendem Template
4. **Bei Bedarf wiederholen** - SOPs sind wiederverwendbar

## SOP vs. Prozess

| Prozess | SOP |
|---------|-----|
| Übergreifender Workflow | Konkrete Anleitung |
| Mehrere Phasen/Agenten | Ein Fokus-Bereich |
| Strategisch | Taktisch |
| "Was und Warum" | "Wie genau" |

SOPs sind die **konkreten Bausteine**, die in Prozessen verwendet werden.
