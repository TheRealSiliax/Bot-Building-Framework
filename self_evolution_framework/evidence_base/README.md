# Wissenschaftliche Evidenz-Basis

## Übersicht

Diese Dokumentation enthält die wissenschaftlichen Grundlagen, auf denen das Self-Evolution Framework basiert. Alle Interventionen und Strategien sind evidenzbasiert und durch Forschung gestützt.

## Evidenz-Standards

Das Framework folgt diesen Standards:

### Evidenz-Hierarchie
1. **Meta-Analysen & Systematische Reviews** (höchste Priorität)
2. **Randomisierte kontrollierte Studien (RCTs)**
3. **Kohortenstudien**
4. **Fallstudien**
5. **Expertenmeinungen** (niedrigste Priorität)

### Qualitätskriterien
- Peer-reviewed Publikationen bevorzugt
- Replizierte Befunde haben Vorrang
- Effektstärken werden angegeben
- Limitationen werden benannt

## Kernkonzepte im Überblick

### Kognitive Verhaltenstherapie (CBT)
- **Effektstärke**: d = 0.8-1.0 (groß)
- **Mechanismus**: Gedanken → Gefühle → Verhalten
- **Anwendung**: Kognitive Umstrukturierung, Verhaltensexperimente

### Akzeptanz- und Commitment-Therapie (ACT)
- **Effektstärke**: d = 0.6-0.7 (mittel bis groß)
- **Mechanismus**: Psychologische Flexibilität
- **Anwendung**: Akzeptanz, Defusion, wertebasiertes Handeln

### Habit Science
- **Kernbefund**: Gewohnheitsbildung ≈ 66 Tage
- **Mechanismus**: Cue → Routine → Reward
- **Anwendung**: Implementation Intentions, Habit Stacking

### Positive Psychologie
- **Effektstärke**: d = 0.3-0.5 (klein bis mittel)
- **Mechanismus**: Ressourcenorientierung
- **Anwendung**: Gratitude, Stärken, Flow

## Disclaimer

Dieses Framework basiert auf wissenschaftlicher Forschung, ersetzt aber **keine professionelle psychologische oder medizinische Behandlung**. Bei klinisch relevanten Symptomen (Depression, Angststörungen, etc.) sollte immer professionelle Hilfe in Anspruch genommen werden.
